# 💀 Login Failed Successfully

> *“If you log in… you win. Good luck.”*

---

## 🤯 What is this?

**Login Failed Successfully** is not your normal login system.
It’s a **chaotic, adversarial UI experience** where the system actively works *against* you.

What starts as a simple login quickly turns into:

* chasing your own username across the screen 🏃‍♂️
* fighting a button that refuses to be clicked 🖱️
* surviving a password system that keeps getting worse 😈

---

## 🎮 Features

### 🧑 Username… but make it painful

* Every character you type **escapes the input box**
* You must **catch and reassemble your own name**

---

### 🖱️ The “Get Started” Button

* Runs away from your cursor 😭
* Has a **hidden timer**
* Only becomes clickable after multiple failed attempts

---

### 🔐 The Password System (Main Boss Fight)

* Rules are **hidden**
* Revealed **one by one**
* Each rule:

  * Must be satisfied
  * Or you face **infinite error popups 💀**

---

### 💥 Chaos Engine

Every 10 seconds:

* Screen flashes neon 🎨
* UI rotates / shifts
* Random popups appear
* Everything becomes slightly worse

---

## 🧠 Concept

This project explores:

* **Bad UX by design**
* **Dark patterns**
* **User frustration as gameplay**
* “What if the system *hated* you?”

---

## 🏆 Why this exists

Because normal login forms are boring.

This project asks:

> *How far can we push bad design… and still make it fun?*

---

## 🚀 How to Run

```bash
1. Download the project
2. Open index.html in your browser
3. Try to log in (you won’t)
```
